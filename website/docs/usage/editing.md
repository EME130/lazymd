# Editing

## Entering Insert Mode

| Key | Action |
|-----|--------|
| <kbd>i</kbd> | Insert before cursor |
| <kbd>a</kbd> | Insert after cursor |
| <kbd>I</kbd> | Insert at beginning of line |
| <kbd>A</kbd> | Insert at end of line |
| <kbd>o</kbd> | Open new line below |
| <kbd>O</kbd> | Open new line above |

## Deletion

| Key | Action |
|-----|--------|
| <kbd>x</kbd> | Delete character under cursor |
| <kbd>dd</kbd> | Delete entire line |
| <kbd>Backspace</kbd> | Delete character before cursor (Insert mode) |
| <kbd>Delete</kbd> | Delete character at cursor (Insert mode) |

## Undo / Redo

| Key | Action |
|-----|--------|
| <kbd>u</kbd> | Undo last change |
| <kbd>Ctrl+R</kbd> | Redo |
