# Editing Modes

lazy-md uses vim-style modal editing with three modes:

## Normal Mode

The default mode. Used for navigation, commands, and switching to other modes. The status bar shows **NORMAL**.

## Insert Mode

For typing text. Enter with <kbd>i</kbd>, <kbd>a</kbd>, <kbd>o</kbd>, or <kbd>O</kbd>. Exit with <kbd>Escape</kbd>. Status bar shows **INSERT**.

## Command Mode

For executing commands. Enter with <kbd>:</kbd>. Status bar shows **COMMAND**.
