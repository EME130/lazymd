# Commands

Enter Command mode by pressing <kbd>:</kbd> in Normal mode.

| Command | Action |
|---------|--------|
| `:w` | Save file |
| `:q` | Quit (warns if unsaved changes) |
| `:q!` | Force quit without saving |
| `:wq` or `:x` | Save and quit |
| `:w <path>` | Save as a different file |
| `:e <path>` | Open a different file |
