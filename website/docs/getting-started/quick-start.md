# Quick Start

```bash
# Open a markdown file
lazy-md README.md

# Open in current directory (shows file tree)
lazy-md

# Create a new file
lazy-md notes.md
```

Once inside the editor:

1. Press <kbd>i</kbd> to enter Insert mode
2. Type your markdown content
3. Press <kbd>Escape</kbd> to return to Normal mode
4. Type <kbd>:w</kbd> and <kbd>Enter</kbd> to save
5. Type <kbd>:q</kbd> and <kbd>Enter</kbd> to quit
